🌙 다크 테마 (기본)
현재 기본 테마입니다. 어두운 배경에 블루 포인트 컬러.
🌸 파스텔 핑크
밝고 귀여운 분위기. 핑크 계열의 부드러운 색감.
🌌 다크 블루
세련되고 전문적인 느낌. 어두운 남색 배경.
⚡ 네온 그린
사이버펑크 감성. 검정 배경에 강렬한 네온 그린.
🍊 오렌지 에너지
따뜻하고 활기찬 분위기. 오렌지와 브라운 조합.

📝 테마 적용 방법

  1. 위에서 원하는 테마의 "적용하기" 버튼을 클릭하세요
  2. CSS 코드가 클립보드에 복사됩니다
  3. styles.css 파일을 엽니다
  4. 맨 위의 :root { ... } 부분을 찾습니다
  5. 복사한 코드로 교체합니다 (또는 파일 끝에 추가)
  6. 브라우저를 새로고침하면 테마가 적용됩니다!

💡 팁: custom-themes/ 폴더에 각 테마의 전체 CSS 파일이 있습니다. 더 자세한 커스터마이징은 UI_CUSTOMIZATION_GUIDE.md를 참고하세요!
